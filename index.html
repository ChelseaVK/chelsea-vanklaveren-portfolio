Recipe Generator

The AI Recipe Generator is a sleek and user-friendly web-based tool designed to transform everyday ingredients into delicious, personalized recipes. Built using HTML, CSS, JavaScript, and powered by AI through an intuitive API, this application blends smart technology with culinary creativity. Users simply enter the ingredients they have on hand, and the tool intelligently suggests recipe ideas tailored to what’s available. Its clean interface ensures a seamless experience, making meal planning both effortless and inspiring—perfect for home cooks, busy professionals, or anyone craving a tasty spark in the kitchen.

Forecast

I built a responsive weather app that delivers a five-day forecast using real-time API data. Beyond just temperature, it updates live with details like humidity and wind speed, keeping users informed and prepared. Interactive elements—like dynamic weather icons that shift with the conditions—add an engaging layer to the experience. Designed with HTML, CSS, and JavaScript, the app emphasizes user-centered functionality wrapped in a clean, intuitive interface.

World Clock App

This world clock app may look simple, but handling time zones in JavaScript is no small feat. Using HTML, CSS, and JavaScript, along with the Moment Timezone library and real-time API integration, I built a sleek and modern interface that keeps complexity under the hood. Users can select from dropdown options to check the time around the globe—without distractions or extra frills.
